# Example definition repository

This markdown ships inside `examples/definition-repo/docs/` for local testing and CI. At **MkDocs** build time, `scripts/aggregate_docs.py` copies each definition repo’s `docs/` tree into the framework site under a repo-specific section (see the framework’s developer docs).

Add dataset narratives under `docs/datasets/` and worked examples under `docs/examples/` as your definition repository grows.
